# Interview Drills

## Module 00: Keyboard Drills

- Navigate from one named checkpoint to another without touching the mouse.
- Explain which shortcuts you rely on to insert rows, edit formulas, switch tabs, go to specific cells, and format borders.
- Rebuild a simple block while keeping one hand on the keyboard and one on the number pad.

## Module 01: Excel Foundations

- Rebuild the answer workbook from the practice file without opening the key.
- Explain when you would use `INDEX/MATCH`, `XLOOKUP`, `SUMIFS`, or `COUNTIF`.
- Clean a messy mini-schedule to match the style guide in under ten minutes.

## Module 02: Banking Workflow

- Explain why assumptions, statements, schedules, outputs, and checks should live on different tabs.
- Trace a forecast revenue assumption all the way through EBITDA, cash flow, and ending cash.
- Defend the balance-sheet check and explain what would break it first.

## Module 03: Comparable Companies

- Walk through enterprise value versus equity value.
- Defend the peer set and identify the first name you would cut if the basket needed to tighten.
- Explain how median peer multiples translate into an implied valuation range for TransUnion.

## Module 04: Precedent Transactions

- Explain why premium percentage alone is not enough to compare deals.
- Walk through the bridge from offer price to implied enterprise value.
- Identify which precedents are strategically relevant versus merely numerically convenient.

## Module 05: DCF

- Build WACC from first principles.
- Explain what happens to implied value if WACC rises by 50 bps.
- Explain why D&A and capex should not be forced equal in a real operating model.
- Contrast perpetuity-growth terminal value with exit-multiple framing.

## Module 06: LBO

- Walk through sources and uses without looking at the sheet.
- Explain what actually drives MOIC versus what drives IRR.
- Describe the purpose of mandatory amortization and a cash sweep.
- Explain how returns move when entry leverage changes versus when exit multiple changes.

## Module 07: VBA Automation

- Explain which macro you would use to clean a pasted schedule before a senior review.
- Describe how an audit macro can help separate formulas, hardcodes, and outright errors.
- Explain why macro libraries are most useful when they remove repetitive low-judgment work rather than replace modeling judgment.
