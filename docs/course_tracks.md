# Workbook Selection Guide

This repo teaches one course. The only choice you make is which workbook version you want to use in the larger modeling modules.

- Standard workbook sequence: use the primary workbook filenames with no `_Full` suffix.
- Extended workbook sequence: use the `_Full` workbook versions where they are available.

Both sequences cover the same concept order and the same modeling standards. The extended workbooks add more worksheet architecture and one extra merger-model capstone; they are not a separate or more "real" course.

## Standard Workbook Sequence

Use this sequence when you want:

- a direct first pass through the course
- faster rebuild cycles and easier repetition
- less worksheet overhead while you lock in process and formula flow
- a clean baseline before taking on the larger workbook versions

Recommended order:

1. `modules/00_keyboard_obstacle_course/IB_Excel_Keyboard_Obstacle_Course_Practice.xlsx`
2. `modules/01_excel_foundations/Excel_Foundations_Practice.xlsx`
3. `modules/02_investment_banking_workflow/TRU_Three_Statement_Workflow_Practice.xlsx`
4. `modules/03_comparable_companies/Information_Services_Trading_Comps_Blank.xlsx`
5. `modules/04_precedent_transactions/Data_Workflow_Precedent_Transactions_Blank.xlsx`
6. `modules/05_dcf_model/TRU_DCF_Practice.xlsx`
7. `modules/06_lbo_model/DNB_LBO_Practice.xlsx`
8. `modules/07_vba_automation/Banker_Productivity_Macro_Demo.xlsx`

## Extended Workbook Sequence

Use the `_Full` workbooks in modules `02`, `05`, and `06`, then add the merger model in module `08`.

Use this sequence when you want:

- the same lesson flow with more worksheet coverage in the major builds
- extra balance, bridge, and support tabs while you model
- a longer timed-build experience in the modeling modules
- an added merger-model capstone after the LBO

Recommended order:

1. `modules/00_keyboard_obstacle_course/IB_Excel_Keyboard_Obstacle_Course_Practice.xlsx`
2. `modules/01_excel_foundations/Excel_Foundations_Practice.xlsx`
3. `modules/02_investment_banking_workflow/TRU_Three_Statement_Workflow_Full_Practice.xlsx`
4. `modules/03_comparable_companies/Information_Services_Trading_Comps_Blank.xlsx`
5. `modules/04_precedent_transactions/Data_Workflow_Precedent_Transactions_Blank.xlsx`
6. `modules/05_dcf_model/TRU_DCF_Full_Practice.xlsx`
7. `modules/06_lbo_model/DNB_LBO_Full_Practice.xlsx`
8. `modules/08_merger_model/SPGI_TRU_Merger_Model_Full_Practice.xlsx`
9. `modules/07_vba_automation/Banker_Productivity_Macro_Demo.xlsx`

## What Changes Between Workbook Versions

- `02` workflow: the extended workbook adds `Build_Map`, `CF`, `BS`, and a balance check.
- `05` DCF: the extended workbook adds `WACC`, `TV_Bridge`, and `Football_Field`.
- `06` LBO: the extended workbook centralizes assumptions and links a fuller debt and returns build.
- `08` merger model: this module sits after the extended sequence as a separate strategic-M&A capstone.

## How To Choose

- Start with the standard workbook sequence if you want the fastest clean run through the course.
- Start with the extended workbook sequence if you want the larger workbook versions from the outset.
- Move from standard to extended once you can finish the primary workbook sequence without losing the model flow.
