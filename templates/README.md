# Templates

Reusable workbook shells for starting a case from scratch without rebuilding the same workbook scaffolding every time.

## Included Template

- `IB_Model_Shell_Template.xlsx`

## What The Template Contains

- cover and instructions tabs
- core `Inputs`, `Calculations`, `Outputs`, and `Checks` tabs
- consistent merged headers and workbook formatting
- a clean banker-style shell you can extend with company-specific tabs

Use the template when you want a banker-style shell before adding company-specific data.
