# 07. VBA Automation

## Objective

Add practical macros that save time in banker workflow: cleanup, trimming, schedule duplication, sensitivity shell setup, audit reporting, and chart export.

## Files

- `Banker_Productivity_Macro_Demo.xlsx`
- `../../vba/modFormatCleanup.bas`
- `../../vba/modSensitivityTables.bas`
- `../../vba/modAuditChecks.bas`
- `../../vba/modScheduleTools.bas`
- `../../vba/modChartExport.bas`

## Workbook Map

- `Macro_Map`: what each module does
- `Raw_Data`: working tab for cleanup, formatting, and audit routines
- `Formatted_Output`: visual target/output tab after cleanup and formatting work
- `Check_Dashboard`: quick readiness view

## Useful Skills Covered

- turning repetitive formatting into one routine
- duplicating schedule blocks without manual copy drift
- setting up a sensitivity shell faster
- building a quick audit report before handing a model to someone else
- exporting chart objects without manual right-click repetition

## Completion Standard

- you can import every `.bas` file into a macro-enabled workbook
- each routine runs cleanly against the demo workbook's sample tabs
- you understand what the macro is changing before you use it on a real model

## Suggested Time

- import and first run: `30-45` minutes
- customization pass: `20-30` minutes
