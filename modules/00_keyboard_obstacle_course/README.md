# 00. IB Excel Keyboard Obstacle Course

## Why This Module Exists

In investment banking, Excel speed is a real job skill. Analysts who stay on the keyboard move faster, break flow less often, and clean up models more efficiently under pressure. This module treats that expectation seriously.

## Files

- In this module: `IB_Excel_Keyboard_Obstacle_Course_Practice.xlsx`
- Answer key: `../../answer_keys/IB_Excel_Keyboard_Obstacle_Course_Answer.xlsx`

## What You Do

- play through a sequence of keyboard-only navigation and editing drills
- move across tabs, ranges, and named checkpoints without using the mouse
- complete fill, formula, and formatting sprints under time pressure
- finish with a multi-sheet boss fight that combines navigation, formulas, and reconciliation

## Workbook Structure

- `Shortcut_Map`: the small set of shortcuts that matter most in finance work
- `Level_01_Navigation`: cell jumping, region movement, sheet switching, and selection control
- `Level_02_Cleanup`: keyboard-only cleanup of a messy model block
- `Level_03_Formula_Relay`: quick-entry formula drills using relative and absolute references
- `Boss_Fight`: a timed multi-step challenge
- `Scorecard`: track time, penalties, and completion level
- `Checks`: quick completion checks after each run
- `Source_Notes`: shortcut and workflow references

## Rules

- keep the mouse off the desk until you are done
- if you touch the mouse, add a penalty on the scorecard
- use the answer workbook only after a full run or when you need to diagnose a stuck section

## Completion Standard

- you can finish levels `01` through `03` cleanly without touching the mouse
- you can explain the shortcuts you used, not just stumble into them
- you can complete the boss fight with no broken checks

## Suggested Time

- first pass: `45-60` minutes
- repeat passes: `20-30` minutes

## Why It Matters For Recruiting

A lot of candidates can talk through valuation. Far fewer can move quickly in Excel without hunting through ribbons. This module is meant to make that difference visible early.
