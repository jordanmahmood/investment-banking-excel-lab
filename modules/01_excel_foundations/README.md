# 01. Excel Foundations

## Objective

Build spreadsheet fluency using real finance data, not toy examples. This module teaches lookups, aggregation, reference locking, charting, and basic formatting discipline in the style expected in banking training programs.

## Files

- In this module: `Excel_Foundations_Practice.xlsx`
- Answer key: `../../answer_keys/Excel_Foundations_Answer.xlsx`

## Workbook Map

- `Raw_Data`: peer trading data and TransUnion quarterly detail
- `Exercises`: guided retrieval, cleanup, aggregation, and formula drills
- `Charts`: chart build and source-table cleanup
- `Checks`: objective completion checks
- `Source_Notes`: supporting references for the dataset and exercise framing

## Useful Skills Covered

- `INDEX/MATCH` or `XLOOKUP` style retrieval
- `SUMIFS`, `COUNTIF`, `AVERAGEIF`, `MEDIAN`, ranking, and spread analysis
- relative versus absolute references
- percent-of-total and margin logic
- keyboard-driven cleanup of ugly pasted blocks
- basic chart setup from a clean source table

## Practice Depth

This is no longer just a handful of blank formulas. The workbook is set up as a sequence of labs:

- guided exercises for slower first-pass work
- a sprint sheet for speed repetition
- a cleanroom block where the answer file acts as a visual formatting benchmark
- checks that force the workbook to reconcile before you move on

## Completion Standard

- every orange practice cell is completed
- the chart source table is clean
- the `Checks` sheet shows `OK` throughout
- your finished practice file resembles the answer file visually, not just numerically

## Suggested Time

- first pass: `60-75` minutes
- rebuild pass without answer key: `30-40` minutes
