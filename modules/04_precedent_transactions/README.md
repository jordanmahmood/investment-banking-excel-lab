# 04. Precedent Transactions

## Objective

Translate public M&A announcements into a banker-style precedent transactions table with premium framing, implied enterprise value, transaction multiples, and a view on deal relevance.

## Files

- `Data_Workflow_Precedent_Transactions_Blank.xlsx`
- `Data_Workflow_Precedent_Transactions_Answer.xlsx`

## Workbook Map

- `Deal_Data`: offer price, shares, equity value, EV bridge, premium-basis notes, and multiples
- `Output`: medians and quick summary conclusions
- `Checks`: population and usability tests
- `Source_Notes`: source-date, premium-basis, and announcement reference support

## Useful Skills Covered

- moving from headline deal language to usable valuation numbers
- separating equity value from enterprise value
- handling premium-basis inconsistency honestly
- recognizing when an EBITDA multiple is not meaningful
- judging which precedents are actually relevant to a case

## Practice Depth

The learner does not just fill formulas. The workbook asks you to reconcile the bridge, classify premium basis, and document why each deal is or is not useful for comparison.

## Completion Standard

- all EV bridges populate
- unusable EBITDA observations are handled cleanly
- output medians calculate without broken logic
- premium-basis and comparability notes are populated with a defendable rationale

## Suggested Time

- first pass: `60-75` minutes
- interview-style walkthrough: `20-30` minutes
